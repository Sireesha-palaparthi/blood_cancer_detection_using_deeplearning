# blood_cancer_detection_using_deeplearning
Deep Learning–based Blood Cancer Detection System that classifies microscopic blood smear images into Benign, Pre, Pro, and Early stages. Evaluated Custom CNN, VGG16, MobileNet, and SqueezeNet models, achieving up to 99.23% accuracy for automated and efficient blood cancer diagnosis.

# Blood Cancer Detection and Stage Identification

## Overview
A Deep Learning-based Blood Cancer Detection System that classifies microscopic blood smear images into four categories:
- Benign
- Pre
- Pro
- Early

The project compares multiple CNN architectures including Custom CNN, VGG16, MobileNet, and SqueezeNet to identify the most effective model for blood cancer classification.

## Features
- Blood smear image classification
- Deep learning-based diagnosis
- Stage identification
- Performance comparison of CNN models
- Automated prediction system

## Technologies Used
- Python
- TensorFlow / Keras
- CNN
- MobileNet
- VGG16
- SqueezeNet
- NumPy
- OpenCV
- Matplotlib

## Dataset
Microscopic blood smear image dataset containing four classes:
Benign, Pre, Pro, and Early.

## Results
| Model | Accuracy |
|---------|---------|
| Custom CNN | 78.22% |
| MobileNet | 95.09% |
| VGG16 | 97.00% |
| SqueezeNet | 99.23% |
